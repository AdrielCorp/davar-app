# Davar — Get the App 🌿

Davar is a companion for reading, studying, and teaching the Bible.

## 📲 Download (Android)

**[⬇️ Download the latest version](https://github.com/AdrielCorp/davar-app/releases/latest)**

1. Tap the link above, then download the `.apk` file.
2. Open it once it finishes. Your phone may ask permission to install apps from this source — tap **Allow**, then go back.
3. Tap **Install**, then **Open**. You're in. 🎉
